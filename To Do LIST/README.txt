Fancy Todo App

This is a fancy and unique Todo App website created using simple JavaScript.

Features

Add tasks with a click of a button.
Delete tasks once they are completed.

How it Works

Enter a task in the input field.
Click on "Add Task" to add the task to your list.
Each task comes with a "Delete" button. Clicking on it will remove the task from the list.
